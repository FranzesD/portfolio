# Personal Portfolio

This repository contains the starter files for a personal portfolio website built with HTML, CSS, and JavaScript.

## Vision

- Showcase your personal brand and skills.
- Present your story, projects, and contact information in a clean, modern layout.
- Deploy easily using GitHub Pages with a single `index.html` entry point.

## What’s Included

- `index.html` — the homepage structure and sections for the portfolio.
- `styles.css` — styling for layout, typography, and responsive behavior.
- `script.js` — simple JavaScript for navigation and interactive enhancements.

## Next Steps

1. Open `index.html` and replace placeholder text with your own name, story, and project details.
2. Add your own color palette, typography, and personal images in `styles.css`.
3. Use GitHub Pages to publish the site from the `main` branch and the root folder.

## GitHub Pages Setup

1. Create a new GitHub repository for your portfolio.
2. Push the contents of this folder to the repository.
3. In GitHub, go to Settings > Pages.
4. Select the `main` branch and root folder (`/`).
5. Save the settings and wait for your portfolio URL.

## Notes

- This starter is intentionally lightweight and easy to extend.
- We can now build the page structure, animations, and project sections together.
